HTML5 Tutorial

📖 42 Chapters ⏱ 6 hours ⭐ 4.9 (12K reviews) 👨‍🎓 18K students

HTML stands for HyperText Markup Language. It is the standard language for creating web pages. Every web page you see is built using HTML. This tutorial covers everything from the absolute basics to advanced HTML5 features like Canvas, SVG, Web Storage, and Semantic Elements.

What is HTML?

HTML (HyperText Markup Language) is the backbone of every website. It defines the structure and content of web pages using a system of tags and attributes. Browsers like Chrome, Firefox, and Edge read HTML files and render them as visual web pages.

HTML works alongside CSS (for styling) and JavaScript (for interactivity) to create the complete web experience you see today.

ℹ️ Did You Know? HTML was first created by Tim Berners-Lee in 1991. The latest version, HTML5, was standardized in 2014 and introduced powerful new features like audio/video support, canvas drawing, and offline storage.

Your First HTML Page

Every HTML document starts with a DOCTYPE declaration followed by the <html> root element. Here's the minimum structure:

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, World! 👋</h1>
  <p>Welcome to my first web page.</p>
</body>
</html>
🖥️ Browser Output
Hello, World! 👋
Welcome to my first web page.

HTML Elements & Tags

An HTML element consists of a start tag, content, and an end tag:

HTML
<!-- Headings (h1 is largest, h6 is smallest) -->
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

<!-- Paragraph -->
<p>This is a paragraph of text.</p>

<!-- Links -->
<a href="https://example.com">Click Here</a>

<!-- Images -->
<img src="photo.jpg" alt="Description">

<!-- Lists -->
<ul>
  <li>Item One</li>
  <li>Item Two</li>
</ul>

Semantic HTML5 Elements

HTML5 introduced semantic elements that clearly describe their meaning to both the browser and developer:

  • <header> – Page or section header
  • <nav> – Navigation links
  • <main> – Main content area
  • <article> – Independent content
  • <section> – Thematic grouping of content
  • <aside> – Sidebar content
  • <footer> – Page or section footer
💡 Pro Tip: Using semantic HTML improves SEO, accessibility, and makes your code more maintainable. Search engines like Google give higher rankings to pages with proper semantic structure!

HTML Forms

Forms are used to collect user input. They're essential for login pages, registration, search, and more:

HTML
<form action="/submit" method="POST">
  <label for="name">Full Name:</label>
  <input type="text" id="name" name="name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <button type="submit">Submit</button>
</form>

CSS Syntax

A CSS rule consists of a selector and a declaration block containing property-value pairs:

CSS
/* Basic CSS Syntax */
selector {
  property: value;
  another-property: value;
}

/* Example */
h1 {
  color: #04AA6D;
  font-size: 2rem;
  font-weight: bold;
}

.my-class {
  background-color: #1c2130;
  padding: 20px;
  border-radius: 12px;
}

Flexbox Layout

Flexbox is a powerful CSS layout module that makes it easy to align and distribute elements:

CSS
.container {
  display: flex;
  justify-content: center;   /* horizontal alignment */
  align-items: center;      /* vertical alignment */
  gap: 16px;                 /* space between items */
  flex-wrap: wrap;           /* allow wrapping */
}

CSS Grid

CSS
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}

/* Responsive grid */
.responsive-grid {
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
}
💡 CSS Custom Properties (Variables) make your styles reusable. Define once, use everywhere: :root { --primary: #04AA6D; } then use color: var(--primary).

Arrays

An array is a collection of elements stored at contiguous memory locations. It's the most fundamental data structure.

JavaScript
// Create an array
const arr = [10, 20, 30, 40, 50];

// Access elements - O(1)
console.log(arr[0]);  // → 10

// Binary Search - O(log n)
function binarySearch(arr, target) {
  let left = 0, right = arr.length - 1;
  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    if (arr[mid] === target) return mid;
    arr[mid] < target ? left = mid + 1 : right = mid - 1;
  }
  return -1;
}
⚠️ Time Complexity: Always analyze your algorithm's time and space complexity. Use Big-O notation: O(1) constant, O(log n) logarithmic, O(n) linear, O(n²) quadratic.

SQL Tutorial

Basic SQL Queries

SQL
-- Select all records
SELECT * FROM students;

-- Select specific columns with condition
SELECT name, age, grade
FROM students
WHERE age > 18
ORDER BY grade DESC;

-- JOIN two tables
SELECT s.name, c.course_name
FROM students s
INNER JOIN enrollments e ON s.id = e.student_id
INNER JOIN courses c ON e.course_id = c.id;

Classes & Objects

Python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} makes a sound"

class Dog(Animal):
    def speak(self):
        return f"{self.name} says Woof!"

dog = Dog("Rex")
print(dog.speak())  # Rex says Woof!
